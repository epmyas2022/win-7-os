# Windows 7

Desktop de windows  con Vue JS y TypeScript, con funcionalidades de un sistema operativo de escritorio.
El proyecto busca simular un sistema operativo de escritorio, con ventanas, barra de tareas, explorador de archivos y mas.

## Instalación

Versiones requeridas:

1. Node.js >= 18.0.0

Instalar dependencias:

```bash
npm install
```
